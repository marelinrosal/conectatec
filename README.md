## 🚀 Tecnologías Clave
| **Capa**       | **Tecnologías**                          |
|----------------|------------------------------------------|
| Frontend       | Kotlin, XML, Jetpack Compose   |
| Backend        | Supabase (DB), Firebase (Auth/Storage)   |
| Pagos         | Stripe API                               |
| Arquitectura  | MVVM/Clean Architecture                 |

## 🔥 Features Destacadas
- ✅ Alertas con **Firebase Auth**
- 🛒 Pasarela de pagos con **Stripe** 
- 🌐 Sincronización en tiempo real con **Supabase** 
- 🧩 UI flexible con **XML**
